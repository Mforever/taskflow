# TaskFlow - Современный трекер задач

![TaskFlow Preview](https://via.placeholder.com/800x400?text=TaskFlow+Screenshot)

## 📱 О проекте

TaskFlow - это полнофункциональное приложение для управления задачами с современным дизайном, темной темой, статистикой и Drag&Drop. Приложение полностью адаптировано для мобильных устройств.

### ✨ Возможности

- ✅ **JWT авторизация** с httpOnly cookies
- ✅ **CRUD операции** с задачами
- ✅ **Drag & Drop** сортировка задач
- ✅ **Приоритеты** (Высокий/Средний/Низкий)
- ✅ **Дедлайны** с календарем
- ✅ **Фильтрация и поиск** задач
- ✅ **Статистика** с графиками активности
- ✅ **Экспорт данных** (CSV/JSON/HTML)
- ✅ **Темная/Светлая тема**
- ✅ **Адаптивный дизайн** для мобильных устройств
- ✅ **Анимации** и плавные переходы

### 🛠 Технологии

**Frontend:**
- React 18 + TypeScript
- TailwindCSS
- React Router v6
- React Hook Form + Zod
- @dnd-kit (Drag & Drop)
- Recharts (Графики)
- Lucide React (Иконки)
- Sonner (Уведомления)
- Vite

**Backend:**
- Node.js + Express.js
- TypeScript
- MongoDB + Mongoose
- JSON Web Tokens (JWT)
- Bcryptjs
- Express Validator
